# Assets Folder

## QRIS Code Image

Tambahkan gambar QRIS code Anda di sini dengan nama: `qris-code.png`

### Cara Menambahkan QRIS Image:

1. Simpan gambar QRIS code Anda (format PNG atau JPG)
2. Rename menjadi `qris-code.png`
3. Place di folder ini: `frontend/src/assets/qris-code.png`

QRIS image akan otomatis tampil di halaman pembayaran.
